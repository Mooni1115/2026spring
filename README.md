# Startup Idea Coach

A prototype web application that recommends and structures business ideas based on a user's profile.

## Overview

Startup Idea Coach is an early-stage project designed to help users move from vague self-description to concrete business ideas.  
The system does not simply generate random startup ideas. Instead, it is intended to evolve into a structured recommendation system supported by:

- user profile interpretation
- business-type knowledge modules
- validation rules
- future market research integration
- future AI and multi-agent expansion

## Current Features

- User profile input form
- Rule-based profile analysis
- Business idea recommendation
- Business idea selection
- Initial business plan generation

## Project Goal

The long-term goal is to build an AI-assisted business ideation and planning system that:

1. analyzes the user's background, resources, and constraints
2. recommends suitable business directions
3. refines selected ideas into a more concrete business structure
4. validates suggestions instead of making careless claims

## Tech Stack

The project includes a structured knowledge base for future expansion.

### Core

### User Interpretation

### Business Types

1. Clone this repository
2. Open `index.html` in your browser
