# Project Notes

## Project Concept

A system that helps users discover and refine business ideas based on their profile, constraints, and interests.

## Why This Project

Many users struggle at three stages:

1. understanding what kind of business suits them
2. generating realistic business ideas
3. turning an idea into an actionable business structure

This project is designed to support all three stages.

## Design Direction

This is not intended to be a simple chatbot that throws out random ideas.  
It should become a structured recommendation system with:

- knowledge-based reasoning
- validation rules
- future market-research integration
- future multi-agent expansion

## Current Scope

- single HTML prototype
- rule-based recommendation logic
- JSON knowledge base design started

## Next Steps

1. Refactor the prototype to load and use JSON files
2. Separate business-type recommendation logic from UI logic
3. Add real AI generation under validation constraints
4. Add market research module
